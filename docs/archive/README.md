# Archive

This folder contains historical planning material and previous long-form changelog snapshots.

## Folders

- `plans/`: implementation plans and design notes from earlier milestones.
- `superpowers/`: planning/review workflow notes.
- `changelogs/`: old changelog snapshots kept for traceability.

Archived docs are not the active deployment contract. When behavior changes, update the current docs first:

- `README.md`
- `README_EN.md`
- `docs/index.md`
- `docs/architecture.md`
- `docs/spec.md`
- `docs/modules/*.md`
- `docs/changelog.md`

Use archive files only as background context or evidence for older decisions.
