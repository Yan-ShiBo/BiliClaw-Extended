BiliClaw Extended

本地优先的跨平台个人内容理解与推荐系统。它用浏览器扩展采集你自己账号能看到的抖音、B 站、小红书、YouTube、X 等信号，在本机保存事件、向量库和画像，并按需调用本机或服务器 Ollama 模型完成分析。

打开 GitHub 查看 README 文档索引

当前重点

多源画像 抖音、B 站为主，小红书和 YouTube 次要，X 作为补充信号。
两个抖音账号 账号分开登录、分开导入，画像阶段用账号 metadata 合并。
本地向量库 抖音喜欢进入 ChromaDB，默认本机 qwen3-embedding:8b。
服务器 LLM 重度分析按需使用服务器 Ollama，大模型用完后释放显存。

本地入口

后端默认端口是 8420。启动后访问 /setup/ 或 /web。
